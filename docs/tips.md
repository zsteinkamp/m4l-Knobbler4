## Knobbler Tips

### Sliders

- Double tap any slider to return to its default value.
- Show or hide the Current Param slider with the dock-bottom toolbar button.

### Clips & Scenes

- Long press a clip or scene to rename it or change its color.
- Show the sidebar mixer strip on the Clips page and use the track titles to change tracks for a very flexible performance system.

### Mapping

- On the Bluhand page, tap a parameter name to enter mapping mode — it will outline in green. Switch to a Knobbler page and tap any green-highlighted slider to map it. Tap the parameter name again to cancel.
- Use the Device Shortcut buttons to quickly access your 8 most-used devices. Map a device the same way as a parameter — select the device, then tap an unassigned shortcut button.

### Navigation

- On tablets, swipe left or right to move between the two Knobbler pages.
- Keep the Navigation panel open on the Clips or Mixer page to quickly jump around your set. Tap a track name to center the view on that track.
- Use the Cue Points button to access your Arrangement View locators.
- Tap a track name in Mixer or Clips view to select that track.
- Tap the track name under a parameter on a Knobbler page to switch to that track.

### Mixer

- Long press a send or pan slider to open a dialog with larger sliders for all sends and panning.
- Long press the Record Arm button to set that channel to "No Input". This is useful in Arrangement View on MIDI tracks where you have clips already recorded — disable input, press Record, and record automation without overwriting the MIDI clips.
- Enable meters when recording for a really nice monitoring solution.

### Workflow

- Use Racks to build reusable instruments or effects chains, and develop muscle memory around their Macro parameter layout on the Bluhand page.
- Capture MIDI is invaluable for quickly building clips or preserving the magic of the moment.

### Setup

- Consider purchasing an inexpensive (~$30 USD) WiFi base station to bring on the road for a simple, wireless setup.
- Connection trouble? Double-check your firewall, disable any VPN, and ensure Knobbler has Local Network access in your device's security settings. See the [Troubleshooting](./troubleshooting.md) page for more help.

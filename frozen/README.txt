Push 3 Standalone Instructions
====================================================================

- Install the Knobbler4-vXX-P3SA.amxd file to your Push
  1) Put Push 3 in Standalone mode and connect to the computer via USB
  2) Start Ableton Live
  3) Find the Knobbler4-vXX-P3SA.amxd file you unzipped and drag it to the "Push" item in the Live Browser. Hover over Push and then drop it on "User Library".
- Add the Knobbler4-vXX-P3SA.amxd file to your Live Set on the Push 3
  1) Navigate to the Main track (small button to the right of the other track selection buttons)
  2) Press the "+" button next to the jog wheel
  3) Device > Max for Live > Knobbler-vXX-P3SA
- Run the Knobbler app on your touch device.
- Go to the Setup page of the Knobbler app.
  - If your P3SA is on a shared WiFi (e.g. your home WiFi), then tap or enter the hostname 'push.local'.
  - If your P3SA is in Hotspot mode, then tap or enter '192.168.0.1' in the Hostname or IP box.
- Enter 2346 into the port box.
- It should show 'connected' at this point.
- If not, bundle up as much information as you can and send it my way. zack@steinkamp.us
